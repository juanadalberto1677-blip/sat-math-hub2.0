Practice Questions

Question 1: Linear Equations

If 3x + 12 = 24, what is the value of x + 4?

  • A) 4
  • B) 8
  • C) 12
  • D) 16
Correct Answer: B (8)
Explanation: First solve for x. 3x = 12, so x = 4. The question asks for x + 4, so 4 + 4 = 8.

Question 2: Percentages

A laptop is discounted by 20%. If the sale price is $800, what was the original price?

Correct Answer: $1000
Explanation: If discounted by 20%, the sale price is 80% of the original.
0.80 * x = 800
x = 800 / 0.80 = 1000.
